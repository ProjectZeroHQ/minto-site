# Dear Minto WordPress Theme

Dear Minto専用のオリジナルWordPressテーマです。既存の静的サイトを単純移植せず、ブランドの信頼感、商品閲覧、無料診断、記事回遊を一体で再設計しています。

## インストール

1. WordPress管理画面の「外観 → テーマ → 新しいテーマを追加 → テーマのアップロード」へ進む
2. `dist/dear-minto-theme.zip` を選び、「今すぐインストール」
3. 「有効化」
4. 「外観 → メニュー」でメインメニューを必要に応じて登録

有効化時に `/blog/` と `/tools/first-ai-task-check/` を自動作成します。同名ページがある場合は重複作成しません。

## 切替前の注意

- DNSはテーマ・記事・フォームの確認が完了するまで変更しない
- 本番切替前にSSLを有効化し、WordPress URLとサイトURLを `https://www-minto.com` に変更する
- 現在の管理者名が推測されやすい場合は、別の管理者を作成後に旧アカウントを削除する
- バックアップ取得後に公開する

## Version 1.2.9

管理画面を開いた際に、次の記事をWordPressの「下書き」として一度だけ自動登録します。

- ChatGPTの回答がずれる原因7つ 仕事で使える指示に直す方法

既に同じスラッグの記事がある場合は重複作成しません。登録後は管理画面上部の通知から編集画面を開き、アイキャッチと表示を確認して公開します。
